# Agent协作流程

本文档定义了律师事务所招聘Agent团队中10个角色之间的协作关系和工作流程。

## Agent角色概览

| 序号 | Agent名称 | 核心职责 | 输入 | 输出 |
|------|----------|---------|------|------|
| 1 | 需求分析 | 岗位需求分析 | 用户需求 | 岗位需求书 |
| 2 | 简历筛选 | 简历初筛评分 | 岗位需求+简历 | 候选人名单 |
| 3 | 初步沟通 | 意向确认 | 候选人名单 | 沟通记录 |
| 4 | 笔试设计 | 题目设计评分 | 岗位需求 | 笔试方案 |
| 5 | 面试安排 | 时间协调 | 笔试成绩 | 面试安排 |
| 6 | 面试评估 | 能力评估 | 面试记录 | 评估报告 |
| 7 | 背景调查 | 信息核实 | 面试通过名单 | 背调报告 |
| 8 | 决策支持 | 录用建议 | 背调报告 | 决策建议 |
| 9 | 数据分析 | 数据支撑 | 全流程数据 | 分析报告 |
| 10 | 综合判断 | 最终决策 | 全流程信息 | 录用结论 |

## 主流程

### 阶段一：需求与筛选

```mermaid
sequenceDiagram
    participant 用户
    participant 需求分析
    participant 简历筛选
    participant 数据分析

    用户->>需求分析: 提出招聘需求
    需求分析->>需求分析: 分析岗位画像
    需求分析->>数据分析: 查询历史数据
    数据分析-->>需求分析: 返回参考数据
    需求分析->>用户: 输出岗位需求书

    用户->>简历筛选: 提供简历
    简历筛选->>需求分析: 获取岗位需求
    简历筛选->>简历筛选: 简历解析评分
    简历筛选->>用户: 输出推荐候选人名单
```

### 阶段二：沟通与笔试

```mermaid
sequenceDiagram
    participant 初步沟通
    participant 笔试设计
    participant 面试安排

    初步沟通->>初步沟通: 联系候选人
    初步沟通->>初步沟通: 确认意向
    初步沟通->>笔试设计: 提供候选人信息

    笔试设计->>笔试设计: 设计笔试方案
    笔试设计->>面试安排: 提供笔试安排

    面试安排->>面试安排: 协调时间场地
    面试安排->>面试安排: 发送通知
```

### 阶段三：面试与评估

```mermaid
sequenceDiagram
    participant 面试安排
    participant 面试评估
    participant 背景调查

    面试安排->>面试评估: 面试安排信息
    面试评估->>面试评估: 记录面试内容
    面试评估->>面试评估: 多维度评分
    面试评估->>面试评估: 生成评估报告

    面试评估->>背景调查: 面试通过名单
    背景调查->>背景调查: 学历核实
    背景调查->>背景调查: 执业核实
    背景调查->>背景调查: 诚信核查
    背景调查->>背景调查: 生成背调报告
```

### 阶段四：决策与录用

```mermaid
sequenceDiagram
    participant 背景调查
    participant 决策支持
    participant 综合判断
    participant 数据分析

    背景调查->>决策支持: 背调报告
    决策支持->>决策支持: 信息整合
    决策支持->>决策支持: 优劣势分析
    决策支持->>决策支持: 薪酬建议
    决策支持->>综合判断: 决策建议

    综合判断->>综合判断: 多维评估
    综合判断->>数据分析: 获取历史数据
    数据分析-->>综合判断: 返回参考数据
    综合判断->>综合判断: 综合判断
    综合判断->>综合判断: 输出录用结论
```

## 异常处理流程

### 简历不足

```
简历筛选Agent 检测到简历不足
        │
        ▼
通知 需求分析Agent
        │
        ▼
需求分析Agent 建议调整：
  ├── 放宽筛选条件
  ├── 拓展招聘渠道
  └── 调整岗位要求
        │
        ▼
用户决策
```

### 面试不通过

```
面试评估Agent 评估不通过
        │
        ▼
记录评估结果
        │
        ▼
通知 简历筛选Agent
        │
        ▼
简历筛选Agent 提供：
  ├── 其他候选人
  └── 建议重新筛选
```

### 背调异常

```
背景调查Agent 发现异常
        │
        ▼
评估异常严重程度
        │
        ├── 低风险 ──▶ 记录，继续流程
        │
        ├── 中风险 ──▶ 通知 决策支持Agent
        │                    │
        │                    ▼
        │               综合评估后决策
        │
        └── 高风险 ──▶ 通知 综合判断Agent
                             │
                             ▼
                        建议不予录用
```

### Offer被拒

```
综合判断Agent Offer被拒
        │
        ▼
记录拒绝原因
        │
        ▼
通知 决策支持Agent
        │
        ▼
决策支持Agent 分析：
  ├── 薪酬差距
  ├── 其他offer竞争
  └── 候选人顾虑
        │
        ▼
提供备选方案：
  ├── 调整薪酬（如可行）
  ├── 启动备选候选人
  └── 重新评估需求
```

## 协作规范

### 信息传递规范

| 信息类型 | 传递方式 | 格式要求 |
|---------|---------|---------|
| 岗位需求 | 结构化文档 | 岗位需求书模板 |
| 候选人信息 | 结构化数据 | 候选人档案模板 |
| 评估结果 | 评分+文字 | 评估表模板 |
| 决策建议 | 结构化报告 | 决策建议书模板 |

### 时间要求

| 环节 | 标准时效 | 最长时效 |
|------|---------|---------|
| 需求分析 | 1个工作日 | 2个工作日 |
| 简历筛选 | 2个工作日 | 3个工作日 |
| 初步沟通 | 3个工作日 | 5个工作日 |
| 笔试安排 | 1周内 | 2周内 |
| 面试评估 | 面试后1日 | 面试后2日 |
| 背景调查 | 3-5个工作日 | 7个工作日 |
| 综合判断 | 1个工作日 | 2个工作日 |

### 质量标准

| Agent | 质量指标 | 目标值 |
|-------|---------|--------|
| 需求分析 | 需求准确率 | ≥95% |
| 简历筛选 | 筛选准确率 | ≥80% |
| 初步沟通 | 联络成功率 | ≥90% |
| 笔试设计 | 题目有效性 | ≥85% |
| 面试评估 | 评估一致性 | ≥85% |
| 背景调查 | 核实准确率 | 100% |
| 综合判断 | 录用质量 | ≥90% |
