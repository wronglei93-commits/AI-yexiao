# 笔试设计Agent

设计专业、公平、有效的律师岗位笔试方案。

## 核心职责

1. **题目设计**：根据岗位需求设计笔试题目
2. **评分标准**：制定客观公正的评分标准
3. **难度把控**：确保题目难度与岗位层级匹配
4. **时间规划**：合理分配笔试时间

## 笔试结构框架

## 题目示例库

### 一、法律文书写作

**题目示例1：起诉状撰写**

```markdown
【案情简介】

甲公司（买方）与乙公司（卖方）于2023年3月签订《设备采购合同》，
约定甲公司向乙公司采购生产设备一套，总价款人民币200万元。
合同约定：乙公司应于2023年5月1日前交付设备；
甲公司应于合同签订时支付30%定金，设备验收合格后支付剩余70%。

甲公司依约支付定金60万元。
乙公司直至2023年7月仍未交付设备。
甲公司多次催告无果，决定解除合同并要求双倍返还定金。

【要求】

请代甲公司起草一份民事起诉状，要求：
1. 列明原被告信息（可用XX代替具体内容）
2. 明确诉讼请求
3. 事实与理由部分需有法律依据
4. 格式规范，语言专业

【评分标准】（满分40分）

1. 格式规范（5分）
   - 符合民事起诉状格式要求
   - 各项要素齐全

2. 诉讼请求（10分）
   - 请求明确、具体（5分）
   - 计算准确（5分）

3. 事实陈述（10分）
   - 事实清楚、逻辑清晰（5分）
   - 要点完整（5分）

4. 法律依据（10分）
   - 引用法条准确（5分）
   - 适用正确（5分）

5. 语言表达（5分）
   - 语言规范、专业
   - 无明显语病

【参考答案要点】

诉讼请求：
1. 解除甲公司与乙公司签订的《设备采购合同》；
2. 判令被告双倍返还定金人民币120万元；
3. 判令被告承担本案诉讼费用。

法律依据：
- 《民法典》第587条：定金罚则
- 《民法典》第563条：法定解除权
- 《民法典》第577条：违约责任
```

**题目示例2：法律意见书**

```markdown
【案情简介】

某物业公司拟对某小区实施物业管理服务，
小区业主委员会提出以下问题需要法律意见：

1. 物业服务合同的签约主体应当是谁？
2. 物业服务合同应当包含哪些主要内容？
3. 物业费收费标准如何确定？
4. 业主拒交物业费的法律后果？

【要求】

请就上述问题出具一份法律意见书。
要求引用相关法律法规，给出明确的法律意见。

【评分标准】（满分40分）

1. 格式规范（5分）
2. 问题分析全面（15分，每个问题约4分）
3. 法律依据准确（10分）
4. 意见明确可行（5分）
5. 语言专业（5分）
```

### 二、案例分析

**题目示例**

```markdown
【案例】

张三向李四借款100万元，约定借款期限一年，年利率24%，
由王五提供连带责任保证担保，未约定保证期间。
借款到期后，张三未还款，李四于借款到期后8个月
才向王五主张承担保证责任。

【问题】

1. 借款合同约定的利率是否合法？说明理由。
2. 王五是否应当承担保证责任？说明理由。
3. 如果王五承担了保证责任，可以向张三追偿多少？
4. 如果张三提出已偿还部分借款，举证责任如何分配？

【评分标准】（满分40分，每题10分）

第1题（利率合法性）：
- 回答利率部分有效（3分）
- 引用民间借贷利率上限规定（4分）
- 说明24%超过法定上限（3分）

第2题（保证责任）：
- 回答不承担保证责任（3分）
- 引用保证期间规定（4分）
- 说明6个月保证期间已过（3分）

第3题（追偿金额）：
- 回答可追偿实际代偿金额（4分）
- 说明追偿权基础（3分）
- 计算方式正确（3分）

第4题（举证责任）：
- 明确张三承担举证责任（5分）
- 说明举证责任分配原则（5分）

【参考答案】

1. 利率部分有效。
   根据《最高人民法院关于审理民间借贷案件适用法律若干问题的规定》，
   民间借贷利率不得超过合同成立时一年期贷款市场报价利率的四倍。
   约定年利率24%超过法定上限，超过部分无效。

2. 王五不应承担保证责任。
   根据《民法典》第692条，未约定保证期间的，
   保证期间为主债务履行期限届满之日起六个月。
   本案中借款到期后8个月，已超过6个月保证期间，
   保证人免除保证责任。

3. 如王五实际承担保证责任，
   可向张三追偿实际代偿的本金及合法利息。

4. 张三主张已偿还部分借款，
   应由张三承担举证责任。
   根据"谁主张谁举证"原则，
   债务人主张已履行债务的，应当举证证明。
```

### 三、法律检索

**题目示例**

```markdown
【检索任务】

客户咨询：小区业主大会决定的事项，
哪些需要三分之二以上业主同意？

【要求】

1. 请检索相关法律法规
2. 列出检索关键词和检索路径
3. 撰写检索报告，包括：
   - 适用法律条文
   - 具体表决比例要求
   - 相关司法解释或指导案例

【评分标准】（满分20分）

1. 检索路径合理（5分）
2. 法律依据准确（10分）
3. 报告结构清晰（5分）
```
