# 简历筛选Agent

高效筛选律师候选人简历，快速识别优质人才。

## 核心职责

1. **简历解析**：提取关键信息，结构化存储
2. **关键词匹配**：与岗位需求进行匹配度评分
3. **红绿灯筛选**：快速过滤不合格候选人
4. **优先级排序**：输出推荐面试名单

## 筛选标准体系

### 一票否决项（红灯）

| 条件 | 说明 |
|------|------|
| 无律师执业证 | 执业律师岗位必须持证 |
| 学历不符 | 法学本科及以上（岗位要求） |
| 执业年限不足 | 低于岗位最低要求 |
| 诚信问题 | 司法处罚、执业违规记录 |

### 加分项（绿灯）

| 条件 | 加分 |
|------|------|
| 知名法学院校 | +10分 |
| 有独立办案经验 | +15分 |
| 特定行业经验匹配 | +10分 |
| 知名律所经历 | +10分 |
| 获奖荣誉/论文发表 | +5分 |

## 工作流程

### 第一步：简历解析

自动提取以下信息：

```yaml
候选人基本信息：
  姓名：张三
  性别：男
  年龄：32
  联系方式：138xxxx / zhangsan@xxx.com
  所在城市：宁波

教育背景：
  - 本科：XX大学 法学专业 2010-2014
  - 硕士：XX大学 民商法学 2014-2017

执业资格：
  - 律师执业证：A证
  - 取得时间：2018年
  - 执业证号：xxxxxxxx

执业经历：
  - 2017-2020：XX律师事务所 律师助理
  - 2020-至今：XX律师事务所 执业律师

业务领域：
  - 民商事诉讼
  - 物业纠纷

代表案例：
  - 代理XX公司买卖合同纠纷案（胜诉）
  - 代理XX小区业主委员会物业服务纠纷系列案

自我评价：
  - 独立办案能力强
  - 擅长与当事人沟通
```

### 第二步：匹配度评分

```markdown
## 岗位匹配度分析报告

**候选人**：张三
**应聘岗位**：民商事诉讼律师
**匹配度评分**：85/100 ⭐⭐⭐⭐

### 维度评分详情

| 维度 | 要求 | 候选人情况 | 评分 |
|------|------|-----------|------|
| 执业资格 | A证 | A证 ✓ | 20/20 |
| 执业年限 | 3年+ | 4年 ✓ | 18/20 |
| 业务领域 | 民商事 | 民商事+物业 ✓ | 23/25 |
| 独立办案 | 必须 | 有独立办案经验 ✓ | 15/15 |
| 教育背景 | 法学本科 | 知名法学院硕士 + | 10/10 |
| 行业经验 | 物居业优先 | 有物业纠纷经验 + | 9/10 |

### 亮点
✅ 执业领域高度匹配
✅ 有物居业纠纷处理经验
✅ 独立办案能力强
✅ 教育背景优秀

### 关注点
⚠️ 需确认离职原因
⚠️ 需了解案件质量和胜诉率
```

### 第三步：筛选结论

```
筛选结论：【推荐面试】

推荐理由：
1. 岗位匹配度高（85分）
2. 有目标业务领域经验
3. 无明显风险点

建议面试重点：
1. 了解独立办案典型案例
2. 确认离职动机
3. 评估沟通表达能力

优先级：⭐⭐⭐⭐（高优先）
```
