# 案件归档整理工具

自动识别、排序并合并案件归档文件为PDF。

## 文件结构

```
archive-tools/
├── archive/              # 归档文件夹（放入需要整理的PDF）
├── output/               # 输出文件夹（合并后的PDF）
├── logs/                 # 日志文件夹
├── config.json           # 配置文件（文件顺序规则）
├── archive_processor.py  # 主处理脚本
├── reset.py              # 重置脚本
├── start.command         # macOS启动文件（双击运行）
├── reset.command         # macOS重置文件（双击重置）
├── start.bat            # Windows启动文件（双击运行）
├── reset.bat            # Windows重置文件（双击重置）
└── README.md            # 说明文档
```

## 使用方法

### 1. 准备文件
将需要归档的PDF文件放入 `archive` 文件夹

### 2. 运行归档
- **macOS**: 双击 `start.command`
- **Windows**: 双击 `start.bat`

### 3. 查看结果
合并后的PDF将保存在 `output/案卷归档.pdf`

### 4. 重置工具
- **macOS**: 双击 `reset.command`
- **Windows**: 双击 `reset.bat`

重置会清空 `archive`、`logs`、`output` 三个文件夹

## 命名建议

为获得最佳识别效果，建议按以下格式命名文件：
- `1-案件登记审批表.pdf`
- `3-委托代理合同.pdf`
- `7-起诉状.pdf`
- `9-原告证据材料.pdf`
- `14-民事判决书.pdf`

## 依赖安装

如果系统没有安装Python或依赖库：

```bash
# 安装Python后，安装PDF处理库
pip install pypdf
```

## 注意事项

1. 只支持PDF文件
2. 无法识别的文件会自动跳过
3. 合并前会显示文件清单，需要确认
4. 每次运行会生成操作日志
